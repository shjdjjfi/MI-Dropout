# Graph-OT Calibration (GOTC) report

## 改进版方案（只动推理后处理，不动 backbone）
这版把 GOTC 改成了**双向关系传播 + 置信度融合**：
1. `sinkhorn_calibrate`：先做 OT 去 hub。
2. 在 OT 分数上分别构建：
   - query-query 图（左图）传播；
   - target-target 图（右图）传播。
3. 双向传播结果做置信度加权融合（高置信样本保留 OT 排序，低置信样本更依赖关系修正）。
4. 外层迭代执行上面步骤（`outer_iters`）。

## 运行命令
```bash
python tools/gotc_eval.py --n 800 --d 128 --seed 7
```
