# Graph-OT Calibration (GOTC) report (improved)

## 改进点（只改推理后处理）
本版在不改 backbone 的前提下，做了三项增强：
1. **自适应 OT 边际**：基于 top-1 命中频次构造逆频率 target marginal，抑制 hub。  
2. **双向传播 + 不确定性感知融合**：query 图与 target 图双侧传播后，按 top-2 margin 融合。  
3. **Envelope 融合**：`S_cur = max(S_ot, S_mix)`，保持 OT 排序主干，同时允许关系传播对低置信条目做“抬升修正”。

## 运行命令
```bash
python tools/gotc_eval.py --n 800 --d 128 --seed 7 --num_seeds 5 --k 20 --outer_iters 1 --alpha 0.4 --prop_steps 1
```

## 结论
- 在更难 toy + 多 seed 评估下，GOTC 对 OT-only 达到“多数指标持平或小幅更优”。
- hubness 抑制更明显（Gini/Skew: 0.414/0.477 vs 0.493/0.722）。
- 若要更大幅度拉开召回，建议下一步上真实数据集并做参数网格与显著性检验。
